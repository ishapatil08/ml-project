Machine Learning Classification Project
 Project Overview

This project implements a supervised Machine Learning classification model using the Iris dataset. The objective is to train a model to classify iris flowers into different species based on their features.

 Objectives

Load and explore dataset

Preprocess data

Perform train-test split

Apply feature scaling

Train classification model

Evaluate model performance

Technologies Used

Python

NumPy

Pandas

Scikit-learn

 Algorithm Used

Logistic Regression (Supervised Learning Algorithm)

Evaluation Metrics

Accuracy Score

Classification Report

Confusion Matrix
